# Devfolio static site generator

Devfolio is a lightweight static site generator for creating developer portfolios. It transforms your project data into a fast, SEO-friendly website with minimal configuration.

Template engine is [Edge.js](https://edgejs.dev/docs/introduction).

HTML template is based on https://github.com/mmacneil/devfolio.

## Install

```sh
$ npm install
```

## Usage

```sh
$ node index.js
```
